Project Overview

This project demonstrates an end-to-end approach to provisioning NVIDIA GPU servers, covering hardware readiness, OS installation, driver & CUDA setup, validation, diagnostics, and troubleshooting for production AI and HPC environments.

Supported NVIDIA GPUs

NVIDIA H100

LLM training, HPC, AI research workloads using Hopper architecture.

NVIDIA H200

Memory-intensive AI workloads with enhanced HBM capacity.

NVIDIA B200

Blackwell architecture for next-gen AI factories and massive models.

NVIDIA L40S

Enterprise inference, rendering, visualization & VDI workloads.

Provisioning Steps

  • Hardware & BIOS validation (PCIe, Secure Boot, Power)
  • OS installation (Ubuntu / Rocky Linux)
  • NVIDIA driver installation
  • CUDA toolkit setup
  • GPU topology & performance validation

Validation Commands

  • nvidia-smi
  • nvidia-smi -L
  • nvidia-smi topo -m
  • nvcc --version
  • lspci | grep -i nvidia

Diagnostics & Logs

  • dmesg | grep nvidia
  • nvidia-bug-report.sh
  • Thermal, power & driver health analysis
